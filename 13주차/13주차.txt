17: 웹 개발 기초 - Flask

목표
    • 웹 개발의 기본 구조와 HTTP 프로토콜의 동작 원리를 이해.
    • Python Flask를 사용하여 간단한 웹 서버를 생성.
    • 라우팅과 템플릿 렌더링을 활용하여 동적인 웹 페이지 구현.

학습 내용
1. 웹 개발 기본 개념
            • 클라이언트-서버 구조 이해.
            • HTTP 프로토콜의 기본 개념(요청과 응답, HTTP 메서드).
            • HTTP 상태 코드의 의미(200, 404, 500 등).

2. Flask 시작하기
            • Flask 설치 및 기본 프로젝트 생성.
            • 라우팅(@app.route())으로 간단한 URL 처리.
            • 템플릿 렌더링(render_template)과 기본적인 HTML 페이지 표시.

3. Flask에서 요청 처리
            • URL 파라미터 처리(<variable>).
            • 동적인 페이지 생성.
