문제 3. 네트워크 설정 파일에서 IP 주소와 포트 추출 및 변경

서버 네트워크 설정 파일에서 IP 주소와 포트를 추출하여 특정 패턴에 맞게 변경하는 프로그램을 작성하세요.
이 프로그램은 네트워크 설정 파일에서 주어진 패턴의 IP 주소와 포트를 찾아서 새로 지정된 값으로 업데이트합니다.
 

요구사항
1.  네트워크 설정 파일 읽기
     •  read_config(file_path) 함수: 설정 파일을 읽어 각 줄을 리스트로 저장하여 반환.
     •  파일이 없거나 읽기 권한이 없으면 FileNotFoundError 예외 처리 후 오류 메시지 출력.

2.  IP 주소와 포트 추출
     •  extract_ip_port(config_lines) 함수: 정규표현식을 사용해 각 줄에서 IP 주소와 포트를 찾아 리스트로 반환.
     •  IPv4 형식의 IP 주소(예: 192.168.1.1)와 포트(예: 8080)를 찾기.
     •  IP와 포트가 올바른 형식이 아닌 경우 ValueError 예외 처리.

3.  IP 주소와 포트 변경
     •  update_ip_port(config_lines, old_ip, old_port, new_ip, new_port) 함수: 기존 IP 주소와 포트를 새로운 IP와 포트로 변경한 뒤 업데이트된 내용을 반환.
     •  기존 IP와 포트가 존재하지 않으면 예외 처리 후 알림 메시지 출력.

4.  변경된 설정 파일 저장
     •  save_update_config(config_lines, output_file_path) 함수: 변경된 설정 파일을 새로운 파일에 저장.
     •  파일이 이미 존재할 경우 덮어쓸지 여부를 물어보고 잘못된 입력이 들어오면 예외 처리.
 

입력 파일 예시
    •  network_config.txt 파일 내용:
-------------------------------------------------------------------
server1 192.168.1.1:8080
server2 192.168.1.2:8081
server3 192.168.1.3:9090


출력 예시
-------------------------------------------------------------------
변경 전 IP 주소와 포트
- server1 : 192.168.1.1:8080
- server2 : 192.168.1.2:8081
- server3 : 192.168.1.3:9090

변경 후 IP 주소와 포트
- server1 : 10.0.0.1:8000
- server2 : 192.168.1.2:8081
- server3 : 192.168.1.3:9090

변경된 설정 파일이 'updated_network_config.txt'에 저장되었습니다.
